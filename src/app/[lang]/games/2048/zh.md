# 2048 游戏说明

2048 是一个经典的数字合并益智游戏。使用方向键推动棋盘上所有方块。  
当两个数值相同的方块碰撞时，它们会合并成一个数值翻倍的新方块。

## 规则

- 棋盘为 4x4。
- 每次有效移动后，会随机生成一个新方块（大多数是 `2`，少数是 `4`）。
- 当出现 `2048` 方块时获胜。
- 当棋盘已满且没有可合并的相邻方块时失败。

## 操作

- `↑` `↓` `←` `→`：移动方块
- `撤销`：回退一步已完成的操作
- `重置`：重新开始随机局面
- `预设胜利 / 预设失败`：快速加载测试局面

## 小技巧

- 尽量把最大数字固定在角落。
- 让数字按大小有序排列，减少被打乱的概率。
- 学习阶段可以配合撤销功能，复盘并修正高风险操作。
