## Image to PDF Tool Guide

This tool combines multiple images into one PDF. You can drag to reorder images, choose page size, control image fitting, and add page numbers. All image reading, resizing, and PDF generation happens locally in your browser.

### Supported Image Formats

JPG, PNG, and WebP are supported. After upload, the page shows thumbnails, image dimensions, and file size so you can confirm the order before generating the PDF.

### Page Size and Fitting

Page sizes include A4, Letter, A3, A5, and “same as image”. Practical choices:

- A4: useful for scanned files, photo documents, assignments, and attachments.
- Letter: useful for US-style documents.
- Same as image: useful when you do not want a fixed paper canvas around each image.

Image fitting modes:

- Fit whole image: keep the ratio and show the full image without cropping.
- Fit whole image, do not upscale: small images are not enlarged, useful for screenshots and QR codes.
- Fill page, crop if needed: better for photo albums, but image edges may be cropped.

### Large Images

Large images are resized proportionally according to the “maximum side” setting before being written into the PDF. This reduces browser memory usage and helps avoid oversized PDF files.

Choose a larger maximum side and higher quality for clearer output. Choose a smaller side and normal quality for smaller files.

### Page Numbers

When page numbers are enabled, each page shows current page / total pages at the bottom. The tool reserves footer space so page numbers do not cover the image.
