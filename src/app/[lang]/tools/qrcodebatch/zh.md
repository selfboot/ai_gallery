# Excel / CSV 批量二维码生成工具

这个批量二维码生成工具可以直接读取 Excel 或 CSV 数据表，选择一列编码内容和一列名称，一次性生成带标题和编号的二维码卡片，并导出 PNG、SVG 或 PDF。适合餐厅桌牌二维码、商品标签二维码、门店评价二维码、WiFi 二维码、活动签到牌、货架标签和线下物料批量制作。

## 这个批量二维码工具能做什么

- 直接在浏览器里读取 Excel 或 CSV 文件
- 一列作为二维码内容，一列作为显示名称
- 批量生成带标题、编号和统一模板的二维码卡片
- 一键导出 PNG ZIP、SVG ZIP 或可打印 PDF
- 全流程本地处理，不上传表格数据

## 常见使用场景

- 餐厅桌号二维码：桌号名称 + 点餐链接
- 商品标签二维码：商品名 + 商品详情页
- 门店评价二维码：门店名 + Google Review / 大众点评 / 小程序链接
- WiFi 二维码卡片、会议签到牌、展会物料、仓库货架标签
- 运营团队批量生成可打印二维码素材

## 使用步骤

1. 上传 `.xlsx`、`.xls` 或 `.csv` 文件。
2. 选择编码内容列和名称列。
3. 选择是否在每个二维码卡片上显示标题和顺序编号。
4. 调整卡片尺寸、二维码大小、标题字号、颜色和 PDF 页面布局。
5. 一键导出全部 PNG、SVG 或 PDF。

## 本地处理说明

表格解析、二维码绘制、预览生成、ZIP 打包和 PDF 导出都在当前浏览器完成，数据不会上传到服务器。适合处理内部编码、门店运营表、商品表、库存标签表、WiFi 字符串、vCard 文本等对隐私更敏感的批量二维码场景。

如果你需要从 Excel 批量生成二维码、把 CSV 批量转二维码卡片、导出可打印 PDF、制作餐厅桌牌二维码、商品标签二维码、门店评价二维码或其他表格驱动的二维码物料，这个工具会比逐个生成更省时间。
