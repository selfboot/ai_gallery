This online Excel deduplication tool finds duplicate rows by one selected column or a combination of multiple columns. It outputs the deduplicated full rows, duplicate rows, and duplicate value statistics. All processing runs locally in your browser without uploading files to a server.

## How to use

1. Upload an `.xlsx` or `.xls` file.
2. Select the worksheet and one or more columns used as the duplicate key.
3. Choose whether to skip the header row, trim leading/trailing spaces, and keep the first or last duplicate.
4. Click deduplicate, review the summary, and download the result workbook.

## Features

- Deduplicate by a single column or by a multi-column key.
- Skip the first header row while preserving it in the exported workbook.
- Trim leading and trailing spaces to avoid common copy-and-paste differences.
- Download an Excel result with deduplicated rows, duplicate rows, and duplicate key statistics.
