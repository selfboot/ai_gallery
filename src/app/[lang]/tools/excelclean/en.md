This online Excel field cleaning tool helps fix messy spreadsheet data. It can trim spaces, remove all spaces, convert full-width characters to half-width, change letter case, remove line breaks, remove invisible characters, normalize date formats, and validate phone numbers, ID numbers, and email addresses. All processing runs locally in your browser without uploading files to a server.

## How to use

1. Upload an `.xlsx` or `.xls` file.
2. Select the worksheet and the columns you want to clean or validate.
3. Choose cleaning rules, optional date normalization, and validation checks.
4. Click clean, review changed cells and validation issues, then download the processed Excel workbook.

## Features

- Batch trim leading/trailing spaces, remove all spaces, line breaks, and invisible characters.
- Convert full-width characters to half-width characters for mixed-language spreadsheets.
- Convert text to uppercase, lowercase, or title case.
- Detect Excel date serials and common text date formats, then normalize them to one output format.
- Validate phone numbers, ID numbers, and email addresses with a separate validation report sheet.
