# How to Resize Images for Social Media

This online social media image size generator turns one or more source images into platform-ready cover images, thumbnails, sharing images, and profile assets. It supports common sizes for Xiaohongshu, WeChat Official Account covers, Twitter/X, YouTube, and OpenGraph sharing images.

## Supported social media sizes

The tool includes practical preset sizes:

- Xiaohongshu portrait images in 3:4, 4:5, and 1:1 square formats.
- WeChat Official Account main cover and square cover.
- Twitter/X post image, profile header, and profile avatar.
- YouTube video thumbnail and channel banner.
- OpenGraph share image for websites, articles, landing pages, and social previews.

## Cover crop vs contain

Cover crop fills the full target size. It is useful for covers, thumbnails, sharing cards, and images that should look visually strong. If the source image has a different aspect ratio, some edges may be cropped.

Contain keeps the entire source image visible and fills the empty area with a background color. It is better for screenshots, product images, QR codes, certificates, charts, and images where edge content should not be lost.

## Focus and zoom controls

Focus controls decide which part of the image stays visible when cropping. Move horizontal focus left or right when the main subject is not centered. Move vertical focus up or down when important text or faces are near the top or bottom. Zoom makes the main subject larger in the final frame.

## Batch export multiple social sizes

Upload multiple images, select several presets, and export them together. If only one output file is generated, the tool downloads a single image. If multiple images or multiple sizes are selected, it creates a ZIP file. This is useful when preparing a batch of images for Xiaohongshu, WeChat, OpenGraph, and YouTube at the same time.

## Local browser processing

Image loading, previewing, cropping, fitting, and export all run locally in your browser. The image is not uploaded to a server, which makes the tool useful for unpublished campaign images, article covers, product screenshots, personal photos, and private draft designs.

## Common use cases

- Create Xiaohongshu, WeChat, and OpenGraph images from one cover.
- Generate a 1200 x 630 website sharing image.
- Create a 1280 x 720 YouTube thumbnail.
- Prepare Twitter/X post images and profile headers.
- Fit product screenshots into exact social media image sizes.
