# 图片打码模糊工具怎么用

这个在线图片打码工具可以给图片指定区域添加马赛克、模糊、黑块、白块或自定义颜色遮挡。它适合在发布截图、分享照片、提交反馈、整理教程素材或上传社交平台前，遮挡人脸、手机号、邮箱、地址、车牌、订单号、聊天记录、证件信息和其他隐私内容。

## 使用步骤

1. 上传 JPG、PNG 或 WebP 图片。
2. 在图片上拖拽框选需要遮挡的区域。
3. 选中区域后，可以拖动位置，也可以拉动四角调整大小。
4. 在右侧选择打码方式，例如马赛克、模糊、黑块、白块或自定义颜色。
5. 设置导出格式和图片质量，点击导出打码图片。

## 不同打码方式怎么选

马赛克适合聊天截图、后台截图、订单页面、账号信息和表格截图。它能保留图片的大致布局，同时让文字和数字难以识别。

模糊适合人脸、背景人物、屏幕反光、桌面环境和不需要完全硬遮挡的区域。模糊效果更自然，但如果原图分辨率很高，建议提高模糊强度。

黑块、白块和自定义颜色块适合需要彻底遮盖的信息，例如身份证号、手机号、收件地址、银行卡号、合同金额、客户名称和内部系统字段。

## 支持多个隐私区域

一张图片里往往有多个敏感位置，例如头像、昵称、手机号、地址和订单号。你可以添加多个框选区域，每个区域都可以单独拖动和缩放。右侧区域列表可以快速选中某个区域，方便继续调整。

## 本地处理和隐私

图片读取、区域选择、打码处理和导出都在浏览器本地完成，不上传服务器。对于聊天截图、后台截图、客户资料和证件照片，建议在导出后重新检查一遍，确认所有敏感信息都已经遮挡。

## 常见用途

- 给截图里的手机号、地址、订单号打码。
- 给照片里的人脸、车牌或背景人物模糊。
- 给客服聊天记录、社群聊天截图遮挡昵称和头像。
- 给教程截图遮挡账号、Token、后台地址和内部数据。
- 给证件、合同、账单图片遮挡关键号码和金额。
