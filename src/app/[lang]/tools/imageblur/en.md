# How to Blur or Redact Part of an Image

This online image blur and redaction tool lets you cover selected areas with mosaic, blur, black boxes, white boxes, or custom color blocks. It is useful before sharing screenshots, photos, support tickets, tutorials, documents, and social media images that contain private information.

## How to use the tool

1. Upload a JPG, PNG, or WebP image.
2. Drag on the image to select the area you want to hide.
3. Move the selected box or resize it from the corners.
4. Choose a redaction method such as mosaic, blur, black box, white box, or custom color.
5. Choose an export format and image quality, then download the redacted image.

## Which redaction method should you use

Mosaic is a good choice for screenshots, dashboards, order pages, account information, tables, and chat records. It keeps the rough layout visible while making text and numbers hard to read.

Blur works well for faces, background people, reflections, desktop environments, and areas that should be hidden more naturally. For high-resolution photos, use a stronger blur radius.

Black boxes, white boxes, and custom color blocks are better when sensitive information must be fully covered, such as ID numbers, phone numbers, addresses, bank card numbers, contract amounts, customer names, and internal system fields.

## Multiple redaction regions

One image often contains several private areas. You can add multiple selection boxes, move each one, resize each one, and select regions from the list. This is helpful for screenshots with several names, avatars, phone numbers, addresses, or order IDs.

## Local browser processing

Image loading, region selection, redaction, and export run locally in your browser. Your image is not uploaded to a server. Before sharing the final image, review it once more to make sure every sensitive area has been covered.

## Common use cases

- Redact phone numbers, addresses, and order IDs in screenshots.
- Blur faces, license plates, and background people in photos.
- Hide names and avatars in chat screenshots.
- Cover account names, tokens, admin URLs, and internal data in tutorials.
- Redact numbers and amounts in ID, contract, invoice, and billing images.
