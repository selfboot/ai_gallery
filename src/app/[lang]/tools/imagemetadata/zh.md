# 图片元数据查看和修改工具使用说明

图片元数据是保存在图片文件里的附加信息，常见内容包括拍摄时间、相机品牌、相机型号、镜头参数、GPS 位置信息、作者、版权、软件来源、图片描述、颜色配置 ICC Profile、XMP 信息和 IPTC 新闻图片字段等。很多照片、设计图、商品图、摄影作品和网站素材都会带有这些信息。

这个在线图片元数据工具支持上传 JPG、PNG、WebP、AVIF、HEIC、HEIF、TIFF 等常见图片格式，并在浏览器本地尽可能读取 EXIF、GPS、IPTC、XMP、ICC、TIFF、JFIF、PNG IHDR 等可解析字段。你可以直接查看完整字段表，也可以下载 JSON 文件保存元数据，方便归档、排查图片来源、检查照片拍摄信息或分析图片文件结构。

## 怎么查看图片元数据

1. 点击上传区域，选择一张或多张图片，也可以把图片拖到页面中。
2. 在左侧图片列表里选择需要查看的图片。
3. 右侧会显示图片预览、文件格式、文件大小、元数据字段数量和完整元数据表。
4. 如果字段很多，可以在搜索框输入 `GPS`、`Date`、`Camera`、`XMP`、`ICC`、`Copyright` 等关键词快速过滤。
5. 点击“下载元数据 JSON”可以把当前图片的元数据导出为结构化 JSON 文件。

## 怎么修改图片元数据

当前工具支持可靠写回常见 JPEG EXIF 字段，包括图片描述、作者、版权信息、相机品牌、相机型号和原始拍摄时间。修改方式：

1. 上传 JPG 或 JPEG 图片。
2. 选择这张图片，在“常见 EXIF 修改”区域填写需要修改的字段。
3. 点击“导出修改后的 JPEG”。
4. 浏览器会下载一张新的 JPEG 图片，原始图片不会被覆盖。

如果你想移除照片里的 EXIF 信息，可以点击“清除 JPEG EXIF”。这适合在发布图片、上传证件材料、发送照片或分享作品前去掉不需要公开的相机和拍摄信息。

## 为什么不是所有格式都支持写回

查看元数据和修改元数据是两件不同的事。读取时，浏览器可以解析多种图片容器里的 EXIF、XMP、IPTC 或 ICC 信息；但写回时，PNG、WebP、AVIF、HEIC、TIFF 等格式的元数据容器结构差异很大，不同浏览器和图片库支持也不一致。强行写回可能导致图片打不开、元数据丢失或文件结构损坏。

因此这个工具优先保证“查看尽可能完整”，修改写回先支持最常见、兼容性也最稳定的 JPEG EXIF。对于 PNG、WebP、HEIC、AVIF、TIFF，建议先用本工具查看和导出 JSON；如果需要专业级全格式元数据写回，可以结合桌面端 ExifTool 一类工具处理。

## 常见使用场景

- 查看照片拍摄时间、相机型号、镜头信息和曝光参数。
- 检查图片是否包含 GPS 位置信息，避免公开个人位置隐私。
- 给摄影作品补充作者、版权和图片描述。
- 导出图片 EXIF / XMP / IPTC / ICC 元数据，方便归档或技术排查。
- 清除 JPEG EXIF，减少图片中不必要的隐私信息。
- 检查网站素材、商品图、社媒图片是否带有多余元数据。

## 隐私说明

图片读取、元数据解析、搜索、JSON 导出和 JPEG EXIF 写回都在你的浏览器本地完成，文件不会上传到服务器。处理完成后请根据自己的需要下载 JSON 或新的 JPEG 文件。
