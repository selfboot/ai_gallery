This tool merges multiple PDF files into one directly in your browser. All processing happens locally, so your files are not uploaded to any server.

## How to merge PDF files

1. Upload multiple `.pdf` files.
2. Adjust the merge order by drag-and-drop or by using move up/down buttons.
3. Click **Merge X documents** after at least 2 files are ready.
4. The merged PDF is generated locally and downloaded automatically.

## Features

- Merge multiple PDF files with original page content preserved.
- Supports drag-and-drop sorting to customize output order.
- Fully local processing in browser for better privacy.
- Fast output and one-click download.
