This online Excel column merge and split tool helps clean everyday spreadsheet fields. You can merge several columns into one, split one mixed column into multiple columns, or extract name, phone, and address from contact text. Files are processed locally in your browser and are not uploaded to a server.

## Common Use Cases

1. Merge multiple columns into one, such as combining name, phone, and address into a full contact field.
2. Split one column by delimiters, including comma, Chinese comma, space, newline, tab, semicolon, and custom delimiters.
3. Extract name, phone, and address from one mixed contact field by using the phone number position.
4. Skip empty values while merging to avoid repeated delimiters.
5. Trim split parts and ignore empty parts to make exported data easier to use.

## Why This Can Be Better Than Excel Built-In Features

Excel Text to Columns works well for clean data with one fixed delimiter, but real exported data often mixes commas, spaces, line breaks, tabs, and Chinese punctuation. This tool can recognize multiple delimiters at the same time and gives you both input preview and result preview before download.

Excel formulas can also merge columns, but you need to write formulas, handle empty cells, and manage separators manually. This tool turns those practical choices into simple options, such as skipping empty values, trimming fields, naming new columns, and exporting a ready-to-use workbook.

## How To Use

1. Upload an `.xlsx` or `.xls` file.
2. Choose the worksheet and confirm the header row and first 10 rows in preview.
3. Choose merge columns, split one column, or extract name, phone, and address.
4. Select columns and delimiter settings.
5. Run processing, check the result preview, and download the Excel file.
