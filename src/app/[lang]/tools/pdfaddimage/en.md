## How To Add Images To A PDF Online

This tool inserts one or more images as new pages into an existing PDF. It is useful for adding signature pages, stamp pages, logos, screenshots, QR codes, notes, or supporting images to contracts, reports, invoices, and document bundles. Upload a PDF first, then upload JPG, PNG, or WebP images, and choose the page position where insertion should start. Use 0 to insert at the beginning, or use 3 to insert after page 3.

Images are inserted in the current list order, and you can move images up or down before generating the final PDF. Two layout modes are supported. Auto layout places 1 to 4 images on each inserted PDF page based on the image sequence and aspect ratios. One image per page creates a separate new PDF page for every image. WebP images are converted locally to JPG before being embedded, while JPG and PNG images are embedded directly when possible.

## Add Page Numbers To A PDF

The tool can also add page numbers to the PDF. Page numbers can be placed at bottom center, bottom right, bottom left, top center, top right, or top left. You can also set the starting page number. Page numbers are written into every page, which is useful for scanned files, contract attachments, document bundles, and PDFs prepared for printing or archiving.

## Local Processing And Notes

All PDF reading, image embedding, page numbering, and export happen locally in your browser. Files are not uploaded to a server. This tool inserts images as new PDF pages; it does not cover existing PDF page content. If you need to stamp or sign directly on top of an existing page, that should be handled by a separate PDF stamping tool.
