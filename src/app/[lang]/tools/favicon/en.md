# Favicon Generator Guide

A favicon is the small website icon shown in browser tabs, bookmarks, search surfaces, mobile home screens, and PWA install prompts. A complete favicon package usually includes more than `favicon.ico`: modern sites often need multiple PNG sizes, `apple-touch-icon.png`, Android icons, `site.webmanifest`, and HTML icon tags.

This online favicon generator accepts PNG, JPG, and WebP images and creates common website icon files locally in your browser. You can choose cover crop or contain mode, transparent or solid background, and padding, then download the full favicon package as a ZIP.

## How to generate favicon.ico and website icons

1. Upload a PNG, JPG, or WebP image.
2. Choose the fit mode. Logos usually work well with cover crop, while full artwork may work better with contain mode.
3. Choose transparent background or a solid background color.
4. Adjust padding so the icon does not touch the edge at small sizes.
5. Click “Generate Favicon”.
6. Download individual files or export the full ZIP package.

## What files are generated

- `favicon.ico`: includes 16×16, 32×32, and 48×48 icons for broad browser compatibility.
- `favicon-16x16.png`, `favicon-32x32.png`, and other PNG sizes for modern browsers and high-density screens.
- `apple-touch-icon.png`: used by iPhone and iPad when a site is added to the home screen.
- `android-chrome-192x192.png` and `android-chrome-512x512.png`: used by Android, Chrome, and PWA installs.
- `site.webmanifest`: provides Web App icon metadata.
- `favicon-html-snippet.txt`: contains icon tags you can add to the website `<head>`.

## What images work best

Favicons are usually displayed at very small sizes such as 16×16 and 32×32. Simple shapes, clear edges, centered subjects, and high contrast work best. Square logos, brand marks, single-letter icons, and clean symbols are usually better than complex photos, long text, thin lines, or busy backgrounds.

Use contain mode with padding if your logo is not square. Use cover crop if you want the icon to fill the square.

## Privacy

Image reading, resizing, cropping, ICO generation, PNG export, and ZIP packaging all run locally in your browser. Your source image is not uploaded to a server.
