This free online PDF watermark tool lets you add text watermarks or image watermarks to one or more PDF files in your browser. You can upload one or more PDFs, choose the watermark type, adjust opacity, rotation, position, layout mode, and page range, then export new watermarked PDF files.

Everything runs locally in the browser without uploading your PDF to a server. It works well for contracts, proposals, invoices, internal documents, design drafts, training materials, and other PDF files that need clear ownership, confidentiality, branding, or usage markings.

### Recommended settings for common cases

- Opacity: usually 10% to 25% works well without making the document hard to read
- Rotation: `-30` to `-45` degrees is common for diagonal confidential or draft watermarks
- Font size: use larger text for strong visibility and smaller text for subtle ownership marking
- Image scale: company logos usually work best at a moderate size rather than covering the whole page
- Single watermark position: center, header area, and bottom-right corner are common choices
- Tiled layout: best for confidential, draft, sample, preview, or internal-use documents

### How to write a custom page range

If you do not want to watermark every page, use the custom page range option.

- Single page: `3`
- Separate pages: `1,4,7`
- Continuous range: `2-6`
- Mixed selection: `1,3,5-8`

Page numbers start at 1. Make sure the page numbers stay within the total page count of the uploaded PDF.

### Why image watermarks can be useful

An image watermark is a better choice when:

- you want to use a company logo
- you already have a transparent stamp or watermark graphic
- you want the watermark to match your brand style
- you need a visual mark instead of plain text

If you only need a quick “Confidential” or “Draft” label, text watermarking is faster. If branding matters more, image watermarking is usually the better option.

### Things to keep in mind

- A watermark is visible page content, not a file permission system
- If the document requires stronger control, combine watermarking with access control and document management rules
- For image watermarks, use a clear PNG, JPG, or WebP file
- If the watermark covers too much text, reduce opacity or choose a smaller size
