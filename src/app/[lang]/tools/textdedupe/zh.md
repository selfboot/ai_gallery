# 文本去重和按行去重工具怎么用

这个在线文本去重工具适合处理名单、关键词列表、URL 列表、邮箱列表、SKU、ID、日志片段、配置项、CSV 单列数据和运营文案素材。你可以把文本粘贴到输入框，工具会按行去重，并实时显示去重后的结果和重复项统计。

所有处理都在浏览器本地完成，不会把文本上传到服务器。对于内部客户名单、账号数据、接口返回、日志和未发布内容，本地去重更适合日常整理。

## 支持哪些去重方式

- 按行去重：每一行作为一个独立项，重复行只保留一次。
- 去首尾空格：把 ` apple ` 和 `apple` 视为同一行。
- 去空行：清理复制粘贴时产生的空白行。
- 大小写敏感：可以选择 `Apple` 和 `apple` 是否算重复。
- 保留顺序：默认保留第一次出现的顺序，也可以关闭后按文本排序。
- 重复项统计：显示重复组、重复次数和出现行号，方便排查数据来源。

## 使用步骤

1. 粘贴文本，或上传 `.txt`、`.csv`、`.md`、`.log` 文件。
2. 根据需要开启去首尾空格、去空行、大小写敏感和保留顺序。
3. 查看右侧统计和重复项列表。
4. 复制去重结果，或下载去重后的 TXT。
5. 如果需要审计重复项，可以复制或下载 JSON 报告。

## 常见使用场景

这个工具适合关键词去重、URL 去重、邮箱去重、客户名单去重、商品 SKU 去重、ID 列表去重、日志行去重、接口返回数据去重和文案素材清理。前端开发、数据处理、SEO、运营和内容编辑都经常需要这类按行去重工具。

## 去重口径说明

工具默认会去掉每行首尾空格、删除空行，并保留第一次出现的内容顺序。如果关闭大小写敏感，大小写不同但内容相同的行会被视为重复。重复统计会记录重复项出现在哪些行，便于你判断是否需要回到原始数据继续清洗。
