## 在线给 PDF 添加页码

这个在线免费 PDF 页码工具支持给一个或多个 PDF 批量添加页码。你可以上传 PDF，设置页码位置、显示样式、字号、颜色、粗细、字体和起始页码，然后直接导出新的 PDF 文件。

整个处理过程都在浏览器本地完成，不需要把 PDF 上传到服务器，适合合同、标书、培训材料、课件、报告、回执、归档文件和各种需要统一分页标识的 PDF。对于想在线给 PDF 加页码、给 PDF 插入页码、给 PDF 批量编号、给扫描 PDF 重新编页码的场景，这个工具都比较直接。

### 这个 PDF 页码工具适合做什么

- 给 PDF 批量添加页码
- 给合同、报价单、审批文件补充连续页码
- 给培训资料、课件、打印稿增加页脚页码
- 给指定页码范围添加编号，而不是整份 PDF 全部处理
- 给扫描后重新整理的 PDF 补页码
- 给论文、方案、汇编材料统一页码格式

### 支持哪些页码设置

- 位置：左上、上中、右上、左下、下中、右下
- 样式：纯数字、`Page 1`、`1 / 20`
- 自定义模板：支持 `第 {n} 页`、`第 {n} / 共 {total} 页` 这类格式
- 起始页：支持从任意数字开始
- 字号和颜色：支持常见页码样式调整
- 粗细：支持常规和加粗
- 字体：支持常见中文和英文字体，也支持读取浏览器可访问的本机字体
- 批量处理：支持一次上传多个 PDF
- 指定页面：支持全部页面，或按页码范围处理

### 如何给 PDF 添加页码

1. 上传一个或多个 PDF 文件
2. 选择页码位置
3. 选择页码显示格式
4. 如果需要中文页码或特殊格式，可以选择自定义模板
5. 设置字体、字号、颜色、粗细和起始页码
6. 选择全部页面或指定页面范围
7. 导出加页码后的 PDF

### 指定页面范围怎么写

如果你只想给部分页面添加页码，可以使用常见的页码范围写法：

- `1-3` 表示第 1 页到第 3 页
- `6` 表示只处理第 6 页
- `1-3,6` 表示处理第 1 到 3 页，以及第 6 页
- `2,4,8-10` 表示处理多个单页和一个连续区间

这种方式适合只给正文加页码、不处理封面，或者只给某几页补充编号。

### 自定义页码样式怎么用

如果内置样式不够，你可以用自定义模板设置更符合中文办公习惯的格式。

- `{n}` 表示当前页码
- `{total}` 表示总页数

常见模板示例：

- `第 {n} 页`
- `第 {n} / 共 {total} 页`
- `P-{n}`
- `附件第 {n} 页`

这样可以更方便地给合同、方案、制度文件或内部资料添加符合格式要求的页码。

### 起始页码有什么用

起始页码不是 PDF 的物理页序，而是你想显示出来的编号起点。

例如：

- 起始页码设为 `1`，第一页显示 `1`
- 起始页码设为 `5`，第一页显示 `5`

这适合正文前面已经有封面、目录、说明页，或者你要把多个 PDF 分开编号后再汇总的场景。

### 字体和中文页码说明

如果你需要给 PDF 添加中文页码，比如 `第 1 页`、`第 3 / 共 20 页`，这个工具也支持。你可以直接选择常见的中文字体，或者尝试读取浏览器可访问的本机字体列表。

对于 Windows 和 macOS 常见办公环境，通常可以优先选择这些字体：

- 微软雅黑
- 苹方
- 宋体
- 黑体
- Arial
- Helvetica

如果只是普通英文数字页码，默认字体一般已经足够。

### 常见使用场景

- 论文、报告和说明书加页码
- 合同、报价单和回执加连续页码
- 培训资料、课件和讲义加页脚编号
- 扫描件整理后重新补页码
- 标书、制度汇编、项目方案统一编号
- 导出打印稿前补充页面顺序标识

### 使用建议

- 如果 PDF 原文件页面方向不完全一致，建议先确认页面旋转显示是否正常，再统一加页码
- 如果只想给正文加页码，建议使用指定页码范围，避免封面和封底被加上编号
- 如果需要较正式的打印效果，通常底部居中或右下角更常见
- 如果文档页数较多，`1 / 20` 或 `第 {n} / 共 {total} 页` 会更清楚
