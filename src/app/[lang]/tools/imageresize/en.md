## Batch Resize Images Online

This free online batch image resize tool lets you upload multiple JPG, PNG, or WebP images, apply one shared resize setting, preview the output, and download each resized image or export everything as a ZIP file. It works well for ecommerce product images, blog images, social media assets, avatars, cover images, and any batch image resizing workflow.

- PNG resize tool
- WebP resize tool
- proportional image resize
- exact image size changer

It supports three common resize modes:

- exact width and height output, with either direct stretching or crop-to-fill
- proportional resize based on only one side, width or height
- percentage-based scaling with a draggable resize slider

### Common use cases

- standardize ecommerce product image sizes
- resize blog images to a consistent width
- prepare avatars and cover images in bulk
- scale down large images before uploading to a website or form
- organize large photo exports into web-friendly dimensions

### Exact-size mode

Exact-size mode supports two different behaviors:

- Stretch to target size: directly output the target width and height
- Crop to target size: fill the whole target box and crop the overflow

When crop mode is selected, you can also choose the crop alignment, such as top left, center, or bottom right. This is useful when you want to keep the main subject visible in cover images, thumbnails, banners, or profile images.

### Proportional resize mode

Proportional resize mode keeps the original aspect ratio. You only need to choose one control side:

- resize by width
- resize by height

After you enter one value, the other dimension is calculated automatically. This mode is useful when you want all images to share the same width or the same height without distortion.

### How to use this image resize tool

1. Upload one or more images
2. Choose a resize mode
3. Set the target size or scale percentage
4. Preview the resized output in the image list
5. Download single files or export all resized images as a ZIP

### Supported image formats

This tool currently supports:

- JPG / JPEG
- PNG
- WebP

You can keep the original format or export all files as JPG, PNG, or WebP.

### Privacy and local processing

All image loading, resizing, cropping, preview, and export work happens locally in your browser without uploading files to a server. This makes it practical for private files, design assets, screenshots, product images, and other images you do not want to send to a third-party service.
