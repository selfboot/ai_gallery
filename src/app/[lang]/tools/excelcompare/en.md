This online Excel column comparison tool helps you find values that appear in both columns, only in column A, and only in column B. It supports comparing two columns from one Excel file or one selected column from each of two Excel files. All processing runs locally in your browser without uploading files to a server.

## How to use

1. Choose a comparison mode: two columns in one Excel file, or one column from each of two Excel files.
2. Upload `.xlsx` or `.xls` files, then select the worksheet and columns to compare.
3. Enable skip header row, trim spaces, and containment matching if needed.
4. Click compare to review the three result groups, or download the result workbook.

## Features

- Compare two columns inside one Excel file or compare columns across two Excel files.
- Skip the first header row and trim leading/trailing spaces to avoid common spreadsheet cleanup issues.
- Treat containment as a match, so one cell can match another when either value contains the other.
- Output matched values, A-only values, and B-only values for easy checking and cleanup.
