# PDF 打印方向修复工具

有些扫描版 PDF 在屏幕上看是竖着的，但一打印就变成横的；或者在一个阅读器里看着正常，换个软件就方向不一致。这通常是因为 PDF 依赖页面旋转元数据来显示，而不是把当前看到的方向真正写进页面内容里。

这个工具会把当前可见方向压平进 PDF 本身。处理完成后，新的 PDF 在查看和打印时会更一致，不再依赖打印机或阅读器对旋转信息的不同处理方式。

## 这个工具适合处理什么问题

- 扫描版 PDF 屏幕看着是正的，打印出来却侧着
- 同一个 PDF 在不同软件里方向表现不一致
- 一个扫描包里有些页依赖旋转标记，有些页没有
- 发给客户、学校、法院、档案系统前，想先把打印方向固定下来

## 使用方法

1. 上传 PDF。
2. 工具会读取每一页的尺寸和旋转标记。
3. 重新生成一个把当前可见方向写死的新 PDF。
4. 下载修复后的 PDF，再正常打印即可。

## 本地处理说明

PDF 处理全部在浏览器本地完成，不上传服务器。适合合同、扫描件、证件复印件、诉讼材料、发票、表单等敏感 PDF 文档。
